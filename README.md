## Digital and Computational Demography

This repository contains notes, code, and resources from my learning through the [*Topics in Digital and Computational Demography*](https://www.demogr.mpg.de/en/career_6122/international_advanced_studies_in_demography_6682/courses_6931/topics_in_digital_and_computational_demography_14628/) course offered by the Max Planck Institute for Demographic Research (MPIDR). 


It will grow as I explore this field further, analyze data, and create visualizations — particularly using digital trace, migration, and geospatial data.
